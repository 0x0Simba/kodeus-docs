TAD stands for Tools, Agents, and Data. It is the shared repository that every Kodeus workflow draws from.

## The three kinds of blocks

- **Tools.** MCP connectors to external APIs, data sources, and systems. 4,000+ available out of the box.
- **Agents.** Pre-built or user-published agents with specialized behavior (governance sentinels, market scanners, comms handlers).
- **Data.** Indexed data sources, knowledge bases, and on-chain feeds that any agent can query.

## Why it matters

Without TAD, every agent would have to ship its own tool wiring and data plumbing. With TAD, blocks are reusable, versioned, and attributed.

## Examples

- A **Uniswap swap tool** callable by any DeFi agent
- A **Governance Sentinel agent** forkable into your own DAO workflow
- A **DexScreener feed** accessible to any market agent

Every block in TAD carries lineage through DNAChain. When your block is used by someone else, attribution flows back to you.
