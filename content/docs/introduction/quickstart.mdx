This quickstart takes you from sign-up to a running agent with verifiable execution.

## 1. Sign up

Create a Kodeus account. No installation required. The platform runs in the browser.

## 2. Connect a wallet

Every agent gets its own identity and wallet. Connect your personal wallet once so Kodeus can issue the agent's ERC-8004 credential and x402 payment rail.

## 3. Describe your agent

Open the Creator Agent and describe the outcome in plain language. Example:

> "Track new proposals on DAO X, summarize them, and post the summary to my Telegram every day at 9am."

Write the objective, scope, constraints, tools allowed, and output format. The Creator Agent turns this into a structured workflow.

## 4. Connect a tool via MCP

From the tool catalog, attach the MCP connectors your agent needs (for example Telegram, a DAO feed, or a data source). Kodeus handles auth and rate limits.

## 5. Run and verify

Run the agent in test mode. Review the execution trace in the DNAChain log. Every tool call, input, output, and decision is signed and stored.

Once you are satisfied, promote to production. Your agent now runs autonomously with a full audit trail.
