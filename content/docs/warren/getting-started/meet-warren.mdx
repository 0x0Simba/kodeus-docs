Warren is a private AI fund manager agent inside Kodeus Trading Bay. It understands each user's capital, risk profile, and preferences, then orchestrates institutional-grade trading strategies on their behalf.

Warren covers the full strategy lifecycle:

1. **Recommends strategies** based on market regime, user intent, and real-time signals
2. **Allocates capital** within user-defined guardrails
3. **Executes trades** on connected exchanges using the user's own API keys
4. **Monitors positions** continuously, managing stop-loss, take-profit, and reallocations
5. **Adapts to regime shifts** by nudging users toward better-suited strategies and deploying them with approval

![Meet Warren](/images/warren/meet-warren.png)

## Core principles

- **User-controlled trading.** Funds never leave the exchange. Kodeus requests API access only, no withdrawal permissions.
- **Approval-driven.** Warren recommends and nudges, but deploys only with explicit user approval.
- **Transparent performance.** Every trade is logged with full attribution. On-chain verification where applicable.
- **Guardrails enforced.** Users set max drawdown, daily loss limits, and per-position sizing. Warren operates strictly within those bounds.

## What Warren is not

- Not a custodian. Funds live on Binance, Kraken, OKX, or Bybit.
- Not a black box. Every trade is logged in the portfolio with full attribution.
- Not financial advice. Always review strategies before deploying.
